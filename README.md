# BedMan 🛏️  
*Track your 3D printer beds' usage, wear, and maintenance.*  

## Features  
- ✅ Log **print hours** per bed.  
- 📝 Add **maintenance notes** (scratches, adhesion issues).  
- 🐳 **Docker-ready** with SQLite (no PostgreSQL needed).
